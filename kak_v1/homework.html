Εργασία 1η παράδοση έως 19/3/20

Στην 1η εργασία πρέπει να υλοποιησετε σε java ένα αριθμητικό κουμπιουτεράκι που θα πραγματοποιεί τις πράξεις +,-,*,/.

Εργασία 2η παράδοση έως 19/3/20

Στην 2η εργασία πρέπει να βελτιώσετε το κουμπιουτεράκι. Θα πρέπει να προσθέσετε εξτρά λειτουργίες όπως υπολογισμός ποσοστού, λογαριθμικές πράξεις(με βάσεις 2 ή 10).Επιπλέον θα πρέπει να προσθέσετε το κατάλληλο γραφικο περιβάλλον ώστε να είναι πιο φιλικό προς τον χρήστη.
